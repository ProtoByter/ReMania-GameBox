Detailed Description

template<typename T>
class RMGB::Graphics::queue< T >

The class used for multithreaded render queues.

This is a custom class that represents a FIFO queue, it also uses SDL Mutexes on the PS3 whilst it uses C++11 lock_guard objects on other platforms

Parameters
TThe type of object stored in the queue
